Warning

Payments through sites like Venmo and CashApp will allow the recipient to request other payments in the future. This will happen very rarely for legitimate fundraisers. If you receive multiple payment requests, we advise that you block the requesting account and file a report using the form linked from the menu in the upper left. If we see a pattern of follow-up requests we will remove the fundraiser from the site.
